# EZ-Pages Metatags for Zen Cart 1.5.6 and Later

It's been updated to use Zen Cart v1.5.6c as the change-basis and to move the meta-tags database fields to the `ezpages_content` table.  If your store uses a version of Zen Cart prior to v1.5.6, please continue to use v3.1.1 of this plugin.
