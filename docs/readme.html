EZ-Pages Meta-Tags — for Zen Cart v1.5.1 through v1.5.4

Version 3.0.0

Current Support Thread at Zen Cart Forums: http://www.zen-cart.com/showthread.php?53799-meta-tags-and-title-descriptions-for-EZpages-and-other-pages

What it does

This plugin adds three (3) additional fields in your admin's Tools->EZ-Pages where you can enter the meta title, keywords and description for each page. Values you enter here override the built-in Zen Cart meta-tag processing for that page. The admin-level tool is updated to show the meta-tags' on/off icons depending on whether or not meta-tags have been defined for a given EZ-Page.

Three additional fields are added to your Zen Cart's ezpages table by the plugin's admin-level auto-installer:

  1. pages_meta_title
  2. pages_meta_keywords
  3. pages_meta_description

Installation

This plugin has core-file overwrites and one template-override file; you should always backup your cart's database and files prior to making any changes.

  1. Unzip the plugin's package file, which you've most likely already done if you're reading this!
  2. Rename the /YOUR_ADMIN directory to match your Zen Cart's secret admin-folder name.
  3. Rename the one template-override directory (/includes/modules/YOUR_TEMPLATE) to match your cart's active template folder.
  4. Make a copy of your current /YOUR_ADMIN/ezpages.php file; if you've made previous changes to that file, you'll need to merge those changes with this plugin's updates.
  5. If you've made previous changes to /includes/modules/YOUR_TEMPLATE/meta_tags.php, you'll need to merge those changes with this plugin's updates.
  6. Copy only the plugin's admin-level files to your store's file system.
    1. /YOUR_ADMIN/ezpages.php
    2. /YOUR_ADMIN/includes/auto_loaders/config.zc154_compatibility.php
    3. /YOUR_ADMIN/includes/functions/extra_functions/ezpages_metatags_admin_extra_functions.php
    4. /YOUR_ADMIN/includes/init_includes/init_zc154_compatibility.php
    5. /YOUR_ADMIN/includes/languages/english/ezpages.php
  7. Log into your Zen Cart admin-console; this will cause the auto-installer to run, adding the three new fields to the ezpages database table.
  8. Copy the plugin's store-side file to your store's file system.
    1. /includes/modules/YOUR_TEMPLATE/meta_tags.php
